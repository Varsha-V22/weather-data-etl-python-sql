# Weather Data ETL Pipeline 🌦️

This project demonstrates an ETL pipeline using Python and SQL.

## Tech Stack
- Python
- Pandas
- Requests
- SQLAlchemy
- SQL Server

## Workflow
1. Extract data from Weather API
2. Transform using Pandas
3. Load into SQL database

## How to run
pip install -r requirements.txt
python main.py
